Стандарт кодирования PHP
================================================================================

Используем стандарт кодирования Zend: http://framework.zend.com/manual/ru/coding-standard.html
Но будем придерживаться стандарта symfony для именования файлов и классов.
Например: myWidgetFormUploadFile

## Принципиальные требования

    * Кодировка всех файлов в UTF-8 (без BOM)
    * Перенос строк - LF
    * Отступы - 4 пробела, никаких табов
    * Не оставлять в строке замыкающих пробелов
    * Правильно расставлять фигурные скобки

## Проверка соответствия стандарту

    # Мягкая проверка
    phpcs --standard=Zend -n <имя файла>

    # Строгая проверка
    phpcs --standard=Zend <имя файла>

    # Проверка всего проекта
    phpcs --standard=Zend -n -p --ignore=*/templates/*,*/doctrine/base/* apps lib test config

    # Проверка для любителей писать документирующие комментарии
    phpcs --standard=PEAR <имя файла>
